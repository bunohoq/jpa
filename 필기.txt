JDBC 기반 기술
1. MyBatis
	- Mapping Framework
	- Java <-> Oracle(관계형 DB)
	- 클래스 객체 -> (추가) -> 오라클 레코드

	- ORM
	- Java(객체) <-> 오라클(엔티티)
	- 모든 DB 데이터 > 자바처럼 접근 + 조작 
	- 구현체 > Hibernate
